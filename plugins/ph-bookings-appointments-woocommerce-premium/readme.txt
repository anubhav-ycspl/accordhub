Woocommerce Bookings and appointments from PluginHive is a powerful plugin that allows you to convert your time, products or services etc into a bookable resource. 

== Description ==
This plugin allows service providers and businesses of all types to easily integrate and convert their Woocommerce site with a booking system for the desired products and services. Doctors, lawyers, salon services, dog walkers, accommodation and guest house providers, hotel businesses and rental services could all use this plugin to start their online bookings and appointments in just a few minutes.

This plugin provides the following features : 


1. Convert a woocommerce product or service into a bookable resource.

2. Allows the admin to configure bookings in fixed time slots of minutes or hours.

3. Allows the admin to configure the bookings in fixed number of days or as a single day booking.

4. Allows the admin to configure bookings in months. (Single month or fixed no of months) Customers will be able to select the months that they want to book.

5. Allows the admin to enable a scheduling calendar or date range calendar selection.

6. Display a scheduling calendar that shows available dates and does not allow to book the already blocked slots.

7. All the above options come with availability checks.

8. Display the booking cost per booking slot.

9. Display Total Booking Cost dynamically: When user selects the dates or booking slots, the plugin dynamically shows the booking total cost and the booking start and end dates

What more? the admin configurations are extremely simple and easy to follow, unlike other complex plugins.

This woocommerce extension helps you to start your online bookings in a few clicks and within a few minutes! So empower your business with bookings today. 

== Installation ==
1. Upload the plugin folder to the \'/wp-content/plugins/\' directory.

2. Activate the plugin through the \'Plugins\' menu in WordPress.
